# 📝 Keep Notes Clone

A minimal note-taking web app built with **React**, inspired by **Google Keep**.

## ✨ Features
- ➕ Create new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 📱 Responsive and clean UI
- ⚡ Real-time updates with React state management

## 🛠 Tech Stack
- **Frontend:** React, CSS
- **State Management:** React Hooks
